NutriDiet: AI-Powered Personalized Diet & Health Tracker
======================================================

Project Overview
----------------
NutriDiet is a sophisticated, data-driven web application designed to help users manage their health through personalized nutrition tracking, AI-driven meal planning, and Machine Learning weight forecasting. It integrates advanced technologies to provide a culturally aware and scientifically backed health management system.

Key Features
------------

1. AI Nutrition Coach
   - Powered by Google Gemini AI (2.0 Flash).
   - Provides a natural language chat interface for health advice, recipes, and grocery tips.
   - Context-aware: Understands user's BMR, TDEE, and weight goals during conversations.

2. AI-Powered Indian Diet Planner
   - Generates personalized 1-day meal plans (Breakfast, Lunch, Dinner, Snacks).
   - Specialized for Indian cuisine with support for Vegetarian, Non-Vegetarian, and Vegan diets.
   - Saves daily plans to the database for long-term tracking.

3. Machine Learning Weight Analytics
   - Utilizes Scikit-Learn for linear regression forecasting.
   - Predicts weight trends 7 days into the future based on historical data.
   - Estimates the specific date for reaching target weight goals.
   - Provides statistical metrics (R-Squared, Slope Coefficient) for technical analysis.

4. Smart Hydration Tracking
   - Automatically calculates daily water intake targets based on weight and activity level.
   - Interactive glass-by-glass logging with visual progress indicators.

5. Professional Data Reporting
   - Generates comprehensive health reports in PDF and Excel formats.
   - Supports Weekly, Monthly, and Yearly filtering.
   - Includes an AI-generated executive summary analyzing user performance.

6. Security & User Management
   - Implements Multi-Factor Authentication (MFA) for enhanced account security.
   - Detailed profile management for health metrics (Age, Gender, Height, Weight, Activity Level).

Technology Stack
----------------
- Backend: Python, Django Framework (REST API capability)
- Database: MySQL (Production), SQLite (Development options)
- AI & ML: Google Gemini API, Scikit-Learn, NumPy, Pandas
- Reporting: ReportLab (PDF), OpenPyXL (Excel)
- Frontend: HTML5, CSS3 (Custom Design System), JavaScript
- Authentication: Django Auth System with MFA support

System Architecture
-------------------
The application follows a Model-View-Controller (MVC) architectural pattern:

- Data Layer (Models): Manages User Profiles, Consumption Logs, Weight Records, Water Logs, and Diet Plans.
- Intelligence Layer: Handles AI interactions for diet generation and ML algorithms for weight prediction.
- Reporting Layer: dedicated utilities for generating downloadable document exports.
- API Layer: RESTful endpoints serving frontend requests for profiles, food items, logs, and stats.
